# Tra cứu Thông tư 14/2026/TT-BCA

Ứng dụng HTML tĩnh chạy offline/GitHub Pages, gồm Thông tư chính và các phụ lục đã nhúng sẵn trong file `index.html`.

## Cách upload GitHub Pages

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ các file trong thư mục này lên repository.
3. Vào **Settings → Pages**.
4. Ở **Build and deployment**, chọn **Deploy from a branch**.
5. Chọn nhánh `main`, thư mục `/root`, rồi lưu.

File chính phải giữ tên `index.html`.
